# gh-rust


GitHub extension で、エラーログファイルを、GitHub copilot API を使って、要約します。

## 説明

gh copilot を使って、エラーログを入力した情報を要約します。

## インストール / アンインストール / アップグレード

```sh
gh ext install kouji-sasaya/gh-copilot
```

```sh
gh ext remove kouji-sasaya/gh-copilot
```

```sh
gh ext upgrade copilot
```

## セットアップ

なし

## エラーログをAI 解析する

```sh
gh copilot error.log
```
